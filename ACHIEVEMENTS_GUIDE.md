## ü¶à 1. Pull Shark

**Requirement:** Merge Pull Requests.

- **Bronze:** Merge 2 PRs.
- **Silver:** Merge 16 PRs.
- **Gold:** Merge 128 PRs.

**Strategy:**
Don't spam! Fix typos in documentation, add small features to open source repos, or use our [Speedrun Repo](https://github.com/LuisSambrano/github-achievements-speedrun) to practice valid PRs.

## ü§† 2. Quickdraw

**Requirement:** Close an issue or merge a PR within 5 minutes of it being opened.
**Strategy:**

1. Open an issue on your own repo.
2. Fix it immediately.
3. Close it. (Speed is key!)

## üöÄ 3. YOLO

**Requirement:** Merge a Pull Request without code review.
**Strategy:**

1. Create a branch.
2. Make a change.
3. Merge it to `main` immediately (Admin overrides often trigger this).
   _Warning: Don't do this on critical production apps!_

## üß† 4. Galaxy Brain (Community Choice)

**Requirement:** Get 2 accepted answers in GitHub Discussions.
**Strategy:**

1. Find active repos with a **"Discussions"** tab (e.g., `vercel/next.js`, `supabase/supabase`).
2. Go to the **"Q&A"** category.
3. Find a simple question you can answer accurately.
4. Once the author marks your answer as "Accepted", you are halfway there!

## üëØ 6. Pair Extraordinaire

**Requirement:** Co-author commits.
**Strategy:**
When committing, add this formatting to your commit message (at the bottom, with two new lines):

```
Co-authored-by: Name <email@example.com>
```

## üåü 7. Starstruck (Rare & Free)

**Requirement:** Create a repository that gets 16 stars.
**Strategy:**

1.  Build something useful (like this Badge Scanner!).
2.  Share it on Twitter/LinkedIn.
3.  Add the `hacktoberfest` topic if applicable.
4.  **Pro Tip:** Ask friends to star your repo. It counts!

## üíñ 8. Public Sponsor (The "$1" Badge)

**Requirement:** Sponsor an open source developer.
**Strategy:**

1.  Find a developer you admire who has GitHub Sponsors enabled.
2.  Sponsor them for just **$1** (one-time).
3.  The badge appears on your profile immediately.
4.  _Technically not free, but $1 is cheaper than GitHub Pro ($4/mo)._
