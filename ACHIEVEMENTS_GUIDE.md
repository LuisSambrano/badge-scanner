**Requirement:** Merge Pull Requests.

- **Bronze:** Merge 2 PRs.
- **Silver:** Merge 16 PRs.
- **Gold:** Merge 128 PRs.

**Strategy:**
Don't spam! Fix typos in documentation, add small features to open source repos, or use our [Speedrun Repo](https://github.com/LuisSambrano/github-achievements-speedrun) to practice valid PRs.

**Requirement:** Close an issue or merge a PR within 5 minutes of it being opened.
**Strategy:**

1. Open an issue on your own repo.
2. Fix it immediately.
3. Close it. (Speed is key!)

**Requirement:** Merge a Pull Request without code review.
**Strategy:**

1. Create a branch.
2. Make a change.
3. Merge it to `main` immediately (Admin overrides often trigger this).
   _Warning: Don't do this on critical production apps!_

**Requirement:** Get accepted answers in GitHub Discussions.

- **Bronze:** 2 Answers.

**Strategy:**
Find repositories with active "Discussions" tabs. Look for "Q&A" categories. Answer questions helpfully.
